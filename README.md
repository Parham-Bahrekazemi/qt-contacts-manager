A simple desktop application for managing contacts using Python, PySide6, and SQLite.

## Features

* Add new contacts
* Delete selected contacts
* Remove all contacts
* Store data in SQLite database
* Qt Model/View architecture using QSqlTableModel
* User-friendly graphical interface

## Technologies

* Python 3
* PySide6
* SQLite
* Qt SQL Module
